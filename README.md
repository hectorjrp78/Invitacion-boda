# Invitación de boda

Sitio web sencillo de invitación de boda creado como punto de partida para personalizar contenido, fechas y estilo.

## Archivos principales

- `index.html` – estructura de la invitación.
- `styles.css` – estilos base, diseño responsive.
- `script.js` – comportamiento sencillo del botón RSVP.

## Cómo usar

1. Abre `index.html` en un navegador.
2. Personaliza el texto de la invitación y los detalles.
3. Reemplaza datos de fecha, hora y lugar según tu evento.
